# todo-app-javascript
A modern Todo List application built using HTML, CSS, and JavaScript.  Features include adding tasks, editing tasks, deleting tasks, marking tasks as completed,  task counter, duplicate task prevention, and LocalStorage support for persistent storage.
Features:
• Add new tasks
• Edit existing tasks
• Delete tasks
• Mark tasks as completed
• Task counter
• Prevent duplicate tasks
• LocalStorage support (data persists after refresh)
